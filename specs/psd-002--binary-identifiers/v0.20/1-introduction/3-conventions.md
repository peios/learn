---
title: Conventions
---

This specification conforms to PSD-001.

## Normative keywords

The key words MUST, MUST NOT, SHOULD, SHOULD NOT, and MAY in this
specification are to be interpreted as described in RFC 2119.

## Byte order

All multi-byte integer fields in this specification are little-endian
unless explicitly stated otherwise.

## Notation

Byte offsets and sizes in format tables are in bytes. Hex values use
the `0x` prefix. Byte sequences are written as space-separated hex
pairs: `0a 0b 0c`.
